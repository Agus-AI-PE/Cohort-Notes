##### ==*Lalu, Apa itu JavaScript [Js] ?*==
- bahasa pemprograman yang bisa di-run di browser dan nodejs
	- di browser (runtime) -> empower interactivity dalam website
	- di nodejs (runtime) -> empower backend app; server app; script automation; API tools; dedicated app
	
##### ==*Then, Node JavaScript [NodeJs] ?*==
- runtime yang memperbolehkan menjalankan js di luar browser
- jaman dulu awal2 muncul js hanya bisa run di browser
- nodejs ngasih fitur non js standar -> sehingga bisa untuk membangun backend capabilities.
	- backend capabilities: file-system, network, environment variable, processes
- project nodejs -> code, config, dependencies, environment
  ![Pasted image 20260617194428](https://raw.githubusercontent.com/Agus-AI-PE/notes-media/main/Pasted image 20260617194428.png)

- npm (package manager) -> installer untuk memasang library dari luar ke project
	- setiap install library -> download dari internet/registry ke local disk/project -> tiap butuh lib yang sama di project lain akan download terus-terusan 
- pnpm -> sama, tapi lebih efisien; industri standar
	- lebih efisien nya gimana? setiap install library -> pnpm akan download ke global folder -> di link kalau ada project lain yang perlu library yang sudah pernah dipake
